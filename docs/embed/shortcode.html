ご利用可能なショートコード

[colormeshop_product]
説明
商品情報を表示します
使い方
[colormeshop_product product_id="123" data="price"]
パラメータ
product_id (商品ID)
省略した場合は $_GET['colorme_item'] の値が使われます
data (取得したい属性)
利用可能な属性
  • id (商品ID)
  • name (商品名)
  • model (型番)
  • price (定価)
  • regular_price (通常販売価格 (割引前の販売価格))
  • members_price (会員価格)
  • unit (単位)
  • weight (重量)
  • simple_explain (簡易説明)
  • explain (商品詳細説明)
[colormeshop_image]
説明
商品画像を表示します
使い方
[colormeshop_image product_id="123" type="main"]
パラメータ
product_id (商品ID)
省略した場合は $_GET['colorme_item'] の値が使われます
type (取得したい画像の種類)
利用可能な属性
省略した場合は main になります
  • main (メイン画像)
  • thumbnail (サムネイル画像)
  • other1 (その他画像)
  • ※ 以降、 otherXXX で XXX 番目に登録されている画像を指定してください
  • ※ モバイルアクセス時はモバイル用画像を表示します
[colormeshop_option]
説明
商品オプションを表示します
使い方
[colormeshop_option product_id="123" index=1 data="title"]
パラメータ
product_id (商品ID)
省略した場合は $_GET['colorme_item'] の値が使われます
index (表示するオプション番号)
  • オプションの組み合わせが 6つ ある場合は 1 〜 6 の数字が入ります
  • 省略した場合は 1 になります
data (表示するオプションの属性)
利用可能な属性
省略した場合は title になります
  • title (オプション名)
  • stocks (在庫数)
  • models_number (型番)
  • option_price (販売価格)
  • option_members_price (会員価格)
[colormeshop_cart_button]
説明
カートボタンを表示します
使い方
[colormeshop_cart_button product_id="123" style="washi"]
パラメータ
product_id (商品ID)
省略した場合は $_GET['colorme_item'] の値が使われます
style (ボタンのデザイン)
省略した場合は basic になります
利用可能な属性
  • basic
  • cloth_blue
  • cloth_yellow
  • cloth_green
  • washi
  • check_blue
  • check_red
[colormeshop_page]
説明
テンプレートにレイアウトされている商品ページを表示します
使い方
[colormeshop_page product_id="123" template="default"]
パラメータ
product_id (商品ID)
省略した場合は $_GET['colorme_item'] の値が使われます
template (テンプレート名)
利用可能なテンプレート
  • default (デフォルト)
